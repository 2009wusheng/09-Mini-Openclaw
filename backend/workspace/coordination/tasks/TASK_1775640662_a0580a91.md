# DeepSeek-V3 技术报告核心内容总结

DeepSeek-V3 是 DeepSeek-AI 团队开发的一款强大的混合专家（Mixture-of-Experts, MoE）语言模型，代表了开源大语言模型领域的重要进展。

## 二、核心参数

- **模型规模**：6710亿总参数，每token激活370亿参数
- **训练数据**：在14.8万亿个多样化、高质量的token上进行预训练
- **训练效率**：仅需278.8万H800 GPU小时完成完整训练
- **训练稳定性**：训练过程非常稳定，无不可恢复的损失尖峰，无需回滚

## 三、架构创新

### 1. 高效推理架构
- **Multi-head Latent Attention (MLA)**：在DeepSeek-V2中已验证的高效注意力机制
- 显著降低推理时的计算开销

### 2. 成本效益训练架构
- **DeepSeekMoE**：优化的混合专家架构
- **无辅助损失的负载均衡策略**：创新的负载均衡方法，无需额外的辅助损失函数
- **多token预测训练目标**：增强模型性能的训练策略

## 四、训练流程

2. **监督微调**：针对特定任务进行精细调整
3. **强化学习**：进一步优化模型能力

- **MMLU-Pro**：通用知识理解

- **MATH 500**：数学推理能力
- **AIME 2024**：高级数学竞赛
- **Codeforces**：编程能力
- **SWE-bench**：软件工程任务

### 性能对比

- **媲美领先闭源模型**：性能与主流闭源模型相当

## 六、技术优势

2. **创新性**：无辅助损失负载均衡和多token预测等创新技术
