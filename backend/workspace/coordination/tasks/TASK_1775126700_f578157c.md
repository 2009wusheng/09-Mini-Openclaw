# 香港本周末天气预报 (2026年4月4-5日)

## 总体天气形势

一股低压槽将于明天靠近广东沿岸。周六该区将有骤雨及几阵狂风雷暴，雨势有时较大。该低压槽预料会在下周初于华南沿岸徘徊，天气仍然不稳定。随着低压槽减弱及高空反气旋增强，下周中期广东的骤雨将逐渐减少，天气好转。日间天气炎热。

## 详细天气预报

### 周六 (2026年4月4日)
- **天气状况**：多云，有骤雨及几阵狂风雷暴，部分地区雨势较大
- **温度范围**：22°C - 26°C
- **相对湿度**：75% - 95%
- **风力**：南至东南风3-4级
- **显著降雨概率**：高

### 周日 (2026年4月5日)
- **天气状况**：大致多云，有几阵骤雨及狂风雷暴，稍后骤雨增多
- **温度范围**：23°C - 27°C
- **相对湿度**：75% - 95%
- **风力**：东南风3-4级，稍后5级，高地间中6级
- **显著降雨概率**：中高

## 出行建议

1. **雨具必备**：周六有雷雨，建议随身携带雨伞或雨衣
2. **防风措施**：周日风力增强，特别是高地地区，注意防风
3. **湿度较高**：相对湿度达75-95%，衣物不易干，建议多备换洗衣物
4. **户外活动**：建议安排在室内进行，如需外出请做好防雨准备
5. **交通出行**：雨天路滑，驾车请注意安全，公共交通可能受影响

## 信息来源

- **官方来源**：香港天文台 (Hong Kong Observatory)
- **数据时间**：2026年4月2日更新
